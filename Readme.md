# Grasshopper
![Grasshopper VM](images/grasshopper.jpg) 
## Introduction
Grasshopper is an intentionally vulnerable virtual machine designed for beginner-level ethical hacking practice.
As the name suggests, Grasshopper is easy to catch, making it ideal for users who are new to cybersecurity and penetration testing.

This machine provides a safe environment to learn and practice exploiting common vulnerabilities and misconfigurations.

## Purpose
The main goal of Grasshopper is to help users gain hands-on experience with basic ethical hacking concepts, including:

- Basic Linux commands
- Enumeration and problem-solving skills
- Using and abusing common services
- Open ports and service exploitation
- Simple scripting concepts
- File compression and decompression
- Remote host connectivity (SSH, etc.)

## Vulnerabilities and Services
This machine contains intentionally vulnerable services and open ports that users can enumerate and exploit.

- SSH
- Telnet
- MySQL
- FTP
- HTTP
- Multiple open ports

## Difficulty Level
- Beginner
- Suitable for students and learners
- First-time CTF or lab users

## Installation
1. Download the virtual machine from the provided link.
2. The file is a .`7z` compressed `.ova` file
3. Extract the `.7z` file using tools like:
    - 7-Zip
    - WinRAR
4. Import the extracted `.ova` file into:
    - VirtualBox
    - VMware
5. Start the machine and begin practicing

## Disclaimer
This virtual machine is created strictly for educational and ethical hacking purposes. Do not use the techniques learned here on real systems without proper authorization.
